# 📊 Superstore Sales Analytics & Forecasting Dashboard

## 📌 Overview

This project is an interactive **Sales Analytics & Forecasting Dashboard** built using **Power BI** to analyze retail sales performance and predict short-term trends.

The dashboard focuses on transforming raw data into **clear, actionable insights** that support better business decision-making.

---

## 🚀 Key Features

* Sales, profit, and quantity performance tracking
* Region-wise and segment-wise analysis
* Product category and sub-category insights
* Shipping mode and payment behavior analysis
* State-level sales distribution
* **15-day sales forecasting using time-series analysis**
* KPI cards for quick business overview

---

## 📊 Dashboard Insights

* 💰 Total Sales: **2M**
* 📈 Total Profit: **175K**
* 📦 Total Quantity Sold: **22K**

📌 Key observations:

* **West region** leads with ~33% of total sales, followed by East (~29%)
* **Corporate segment** contributes the highest share (~33%)
* **Cash on Delivery (COD)** is the most used payment method (~43%)
* Sales peak towards **year-end (seasonality trend)**
* High sales do not always translate into high profit (margin gaps)
